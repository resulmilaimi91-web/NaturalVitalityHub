# SI TË PËRDORNI DIGISTORE

## Për Blejuesit:
1. Hyni në http://localhost:8000
2. Kërkojnë produktin
3. Klikoni "Buy Now"
4. Paguani me Paysera
5. Merrni produktin dhe license key

## Për Shitësit:
1. Regjistrohuni si seller
2. Klikoni "Seller Dashboard"
3. Klikoni "New Product"
4. Plotësojnë:
   - Emri i produktit
   - Pershkrimi
   - Çmimi
   - Upload file-in
   - Thumbnail
5. Publikoni

## Për Admin:
1. Hyni me: admin@digistore.com / admin123
2. Keni akses në:
   - Të gjitha produktet
   - Analitika
   - Statistikat
   - Menaxhimi i përdoruesve

## Pagesa:
1. Blejuesi paguan
2. Paratë shkojnë në Paysera
3. Pas 1-3 ditësh në llogarinë bankare
4. Ju merrni notifikim me email

## Kuponat:
1. Shkoni tek Coupon Engine
2. Krijo kuponin
3. Vendosni zbritjen
4. Shpërndajeni kodin

## Affiliate:
1. Regjistrohuni si affiliate
2. Merrni linkun tuaj
3. Shpërndajeni
4. Merrni komision për çdo shitje
